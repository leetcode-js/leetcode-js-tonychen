# LeetCode日常刷题

> 这个仓库主要存放我在 [LeetCode](https://leetcode.com/) 这个网站上拿到``Accepted``的题目

## 分类
大概分成以下三类：

- Easy  
    简单题目：思路简单或者数据量不大，通常可以快速解决,目前``AC``的题目大多属于这类，比较的水
- Medium  
    中等题目 ：目前``AC`` 的数量比较少
- Difficult  
    困难题目：目前``AC`` 的数量比较少

目前能力有限，``AC``的大部分是 [LeetCode](https://leetcode.com/) 上的``Easy``题目，其他类型的题目解法会陆续放到这个仓库
