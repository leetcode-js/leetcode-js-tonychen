## 分类
大概分成以下三类：

- Easy  
    简单的题目：思路简单或者数据量不大，通常可以快速解决,目前``AC``的题目大多属于这类，比较的水
- Medium  
    中等题目 ：目前``AC`` 的数量比较少
- Difficult  
    比较困难的题目：目前还没有``AC``过

目前能力有限，只``AC``了 [LeetCode](https://leetcode.com/) 上的``Easy``题目
